# Bilder & weitere Dokumentation

Hier liegen Screenshots und weiterführende Doku-Dateien, die im Haupt-README referenziert werden.

## Screenshots

- **`screenshot.png`** — wird als Hero-Bild oben im Haupt-README angezeigt.
  Empfohlen: PNG, ca. 1280 × 800 px, zeigt das Dashboard im laufenden Betrieb
  (gerne mit Demo-Daten, wenn keine echte Hardware vorhanden ist).

So legst du den Screenshot ab:

1. Dashboard öffnen (`npm start` oder `RasiCross_Telemetry_v9_6.html` im Browser)
2. Demo-Modus starten (für volle UI ohne Hardware)
3. Screenshot vom Fenster machen (Windows: `Win + Shift + S`)
4. Als `docs/screenshot.png` speichern, committen, pushen

Sobald die Datei da ist, erscheint sie automatisch oben auf der Repo-Seite.
